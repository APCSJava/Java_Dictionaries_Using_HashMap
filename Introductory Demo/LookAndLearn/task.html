Dictionaries

In the casual sense, a dictionary is any structure that creates a mapping from one object to another. The first object, called a key, is used to locate a second object, the value of interest.

Examples

  • An English language dictionary associates an English word with its definition.
  • A Spanish-English dictionary associates a Spanish word with its English equivalent.
  • A dictionary of postal codes, say used by Google Maps, associates a postcode with a street address so that your favorite fast food restaurant courier knows where to deliver your takeaway.

Whenever one object is used to locate another object, the idea of a dictionary is in play.

Dictionaries in Java

Java provides dictionary behavior through the Map interface. The class java.util.HashMap is the most frequently used implementation and the one we will demonstrate here.

Debug the main() method in the file Main.java and notice the effect of the following methods drawn from the HashMap API.

  • HashMap < KeyType, ValueType > dictName

    Declares a variable of type hashmap with the key type and value type specified.

  • new HashMap < > ()

    Calls the constructor. Types can be inferred.

  • put(key, value)

    If the key is not present, adds it to the map and assigns value.

    If the key is already present, updates the value associated with the key.

  • get(key)Returns the value currently associated with the key.
